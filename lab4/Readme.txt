Features :

A simple JavaScript-based quiz program.

Contains multiple basic questions related to general knowledge and web development.

Checks user answers using prompt and shows result using alert.

Calculates total score at the end of the quiz.

Beginner-friendly code written without any libraries or frameworks.

Clean and easy-to-understand logic using loops and arrays.

Inputs are automatically trimmed and compared into lowercase.

Works directly into a browser using a single HTML + JS file.

Can be improved by adding UI, timer, or more questions.
